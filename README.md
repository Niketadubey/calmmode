# Calm Mode Website

This is a simple static website created using HTML, CSS, and JavaScript.  
The purpose of this project is to create a calm and relaxing web interface.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Simple and clean UI
- Calm theme design
- Responsive layout
- Interactive elements using JavaScript

## Project Structure
index.html
project.css
project.js
## How to Run Locally
1. Download or clone the repository
2. Open the folder
3. Open index.html in your browser
